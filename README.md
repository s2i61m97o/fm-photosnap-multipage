### What I learned

First project with Nextjs (App Router). I learnt about routing and file structure including layouts and route groups, useful for creating a layout for a select group of pages. I also learnt about Image and Font optimisation.

I also came across the "currentColor" value for color attributes on SVG's. This allows for better color control with CSS as well as hover effects on SVG's.

### Continued development

I will continue to use Nextjs and dive deeper into its capabilities in future projects, utilising it where possible when needed.

### AI Collaboration

I used AI for 2 main scenarios. If an error is showing which I don't understand or don't know how to fix, I use AI to explain the error for better understanding. Also, if I am either stuck or can think of multiple solutions to a problem, I will use AI to understand the way forward or the pros and cons of each solution, and if there is a solution I haven't thought of.
